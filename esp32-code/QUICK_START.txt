================================================================================
                        REGISTER ESP IN ADMIN PANEL
================================================================================

When simulator shows:
   "‚ö†Ô∏è ESP not registered yet!"
   "üìã MAC Address: AA:BB:CC:DD:EE:09"

Do this:
   1. Open browser ‚Üí http://localhost:3000 (or your frontend URL)
   2. Login to Admin Panel
   3. Click "ESP Devices" in sidebar
   4. Click "+ Register ESP" button
   5. Fill the form:
      - MAC Address: AA:BB:CC:DD:EE:09
      - Drone ID: R1 (or R2-R8, B1-B8)
      - Role: Forward (or Striker/Defender/Keeper)
      - Device Type: ESP32-Dev
   6. Click "Register"
   7. Simulator will auto-detect within 5 seconds!

================================================================================
                        SIMULATION MODES
================================================================================

Edit line 24 in esp32-simulator.py:

   SIMULATION_MODE = "static"   ‚Üí Drone hovering (minimal movement)
   SIMULATION_MODE = "moving"   ‚Üí Circular flight pattern (realistic)
   SIMULATION_MODE = "spinning" ‚Üí Tumbling/crashing (extreme values)

================================================================================
                        COMMON ISSUES
================================================================================

‚ùå "Cannot connect to backend"
   Fix: Make sure backend is running (npm start in backend folder)
        Check SERVER_URL in esp32-simulator.py (line 16)

‚ùå "ModuleNotFoundError: No module named 'requests'"
   Fix: pip install requests

‚ùå Simulator shows registered but no telemetry in match
   Fix: Create match, start round, select registered drones

================================================================================
                        CHANGE BACKEND URL

For LOCAL backend (default):
   Edit line 16: SERVER_URL = "http://localhost:5000"

For DEPLOYED backend (Render):
   Edit line 16: SERVER_URL = "https://your-backend.onrender.com"
